Implicit Chatrooms

As soon as you open GitChat in your repository, it automatically detects and creates a chatroom with your team.

It's as easy as 1 - 2 - 3!

Navigate

Navigate to the repository on your local device, and start GitChat.

Authenticate

Login to GitHub, and enter the chatroom.

Chat

Voila! You may now start chatting to your fellow developers.

Installation

  1. To install GitChat, first ensure that Python is installed on your device.
  2. Next, download GitChat using pip via either of these commands:
    3. pip install gitchat pip install git+https://github.com/shubhodeep9/gitchat.git
  3. Navigate to the repository of your choosing and run the command gitchat. It's as simple as that!
